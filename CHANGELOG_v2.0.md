# HHGC Website v2.0 — Immersive Charcoal Experience

## Visual Direction

- Made charcoal the primary page color for a more modern, hospitality-led experience.
- Retained the approved cream, forest, and gold palette as supporting brand colors.
- Increased use of the HH broken-club icon in the hero, cursor, Impact Lab, and section-scale environmental typography.
- Restyled existing interactive panels, memberships, location, FAQ, and footer to feel like one continuous clubhouse environment.

## Immersive Interaction

- Added a custom branded cursor on supported desktop devices.
- Added a three-screen-length scroll sequence featuring a dimensional golf-club face.
- Connected scroll position to club rotation, ball launch, tracer progress, and example impact metrics.
- Preserved reduced-motion behavior and touch-device fallbacks.

## Content and Asset Safety

- Used only the repository's approved concept renderings, hospitality imagery, and supplied HH logo/icon variants.
- Did not add pricing, an exact address, an opening date, a simulator vendor, presale language, or final House Key terms.
- Kept the existing beer-and-wine atmosphere imagery in place of competitor-source photography.

## Production Note

The repository asset note still identifies some logo PNGs as extractions from the Brand Standards Guide. Replace those prototype files with approved master exports before final public production when the masters are available in the repository.
